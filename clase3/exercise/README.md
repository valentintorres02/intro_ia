## 📋 Descripción del problema
En este ejercicio deberán implementar el algoritmo de **Simulated Annealing** para resolver el problema de **N-Queens**.

El problema de las **N-Queens** es un clásico desafío en informática que consiste en colocar $N$ reinas sobre un tablero de ajedrez de tamaño $N \times N$ de manera que ninguna reina pueda atacar a otra.
